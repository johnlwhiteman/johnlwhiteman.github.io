# Screen

```bash
# Start a session
screen -S johnny

# Disconnect
ctrl+a d

# Show sessions
screen -ls

# Reconnect
ctrl+a r
ctrl+a r <session id>

# Create a new window
ctrl+a c

# Split window vertically
ctrl+a |

# Split window horizontally
ctrl+a S

# Note: Create a new window for every new window
ctrl+a c

# Move to next window
ctrl+a n
ctrl+a tab

# Move to next window
ctrl+a p

# Unsplit window
ctrl+a Q
```
